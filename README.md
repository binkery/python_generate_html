# 简介

一个生成 HTML 静态网站的 Python 脚本。每一个 markdown 文档对应一个 HTML 页面。

## 代码部分

 - main.py
 - config.py
 - template.py

## 使用

1. 修改 config.py 

设置站点名字，站定域名，输入目录，输出目录。

2. 执行

    ./main.py

## 文档约定

在这里，约定 markdown 文档格式，以方便生成 HTML，但其实没有严格遵守的话，从显示上也没有什么特殊的问题。

    # 文档标题
    - 关键字,关键字2,
    - 2019.03.19

结果如下

# 文档标题
- 关键字,关键字2,
- 2019.03.19
